# 📊 Student Attendance Automation (UiPath)

## 🚀 Overview
This project is an end-to-end **Student Attendance Automation system** built using **UiPath (RPA)**.  
It automates the process of reading attendance data, calculating percentages, identifying students below threshold, generating reports, and sending automated email alerts.

The solution reduces manual effort, improves accuracy, and enhances efficiency in academic administration tasks.

---

## 🎯 Features
- 📥 Reads student attendance data from Excel
- 📊 Calculates attendance percentage automatically
- ⚠️ Identifies students below attendance threshold
- 📑 Generates structured reports
- 📧 Sends automated email alerts to students
- ✅ Reduces manual errors and processing time

---

## 🛠️ Technologies Used
- UiPath Studio (Latest Version)
- Excel Automation
- Email Automation (Gmail / Outlook)
- RPA Workflow Design

---
## ⚙️ How It Works
1. The workflow reads attendance data from an Excel file.
2. It calculates attendance percentage for each student.
3. Students below the defined threshold are filtered.
4. A report is generated with categorized data.
5. Automated emails are sent to notify students.

---

## ▶️ How to Run the Project
1. Open UiPath Studio
2. Open the project folder
3. Update the Excel file path if needed
4. Configure email account (Gmail/Outlook)
5. Run `Main.xaml`
